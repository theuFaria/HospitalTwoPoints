<h1>Hospital Two Points</h1>

<p>O Two Points Hospital, está tendo dificuldades quanto ao controle de
medicamentos, e está em busca de implementar um novo sistema de controle, a
fim de adicionar novas funcionalidades e sanar problemas antigos. Por isso, esse projeto é um programa de controle de Estoque de medicamentos para o Hospital sanar suas necessidades.</p>

<h2> Sobre o Projeto</h2>
<p>O objetivo do sistema proposto é organizar o estoque de
medicamentos do hospital, a fim de acabar com a possível falta de medicamentos
e ter um controle preciso sobre o estoque de medicamentos, possibilitando o
rastreio quanto a entrega do mesmo para o paciente. Assim sabendo quais
medicamentos são mais requisitados e necessários. Tendo ferramentas que
possibilitem ver quantidade de remédios em estoque, quantos foram pedidos,
possível data de entrega. Nos casos de retirada de medicamentos, quem retirou,
data e hora.</p>
